# 📝 常见问题解答

## Q1: 如何添加新的材料？
只需要按照上面的`structure_data`格式准备数据即可。

## Q2: 计算需要多长时间？
- 力场计算：几分钟
- DFT计算：几小时到几天
- 电镜模拟：几分钟到几小时

## Q3: 结果保存在哪里？
所有计算结果会自动保存在`cache`文件夹中。
